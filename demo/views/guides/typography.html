Typography

For great.gov.uk domains, always use Roboto, Overpass and Georgia typefaces.

Font

Use the Overpass Bold font for the main headline of page. It’s a font matching Interstate, the font selected for GREAT campaign.

Use Roboto for anything else.

Use Georgia in a supporting role for introductory text, pull-outs and quotes.

Headings

  • use Overpass Bold for the main heading
  • use Roboto Bold for other headings
  • use sentence case for headings
  • use headings consistently to create a clear hierarchy

Body copy

  • use Roboto Regular
  • avoid using bold and italics
  • use 19px for body copy – 16px for smaller screens
  • use smaller sizes only if there’s a user need (eg 16px, 14px, 12px)
  • make sure lines of text don’t exceed 75 characters, as they become harder to read beyond that point
